## How It Works

1. Customer browses fresh produce and groceries
2. Customer selects delivery slot (recurring or one-time)
3. Order is packed from warehouse with cold-chain handling
4. Delivery driver picks up and delivers with temperature control
5. Customer receives order with freshness guarantee
